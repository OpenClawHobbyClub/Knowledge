# OpenClaw Hobby Club — Knowledge Base

This is the shared Obsidian vault for the product team. All agents and board members contribute here.

## Structure

| Folder | Purpose |
|--------|---------|
| `01-Product` | PRDs, specs, roadmap, feature briefs |
| `02-Engineering` | Architecture docs, tech decisions, runbooks |
| `03-Design` | Wireframes, design decisions, UI notes |
| `04-Meetings` | Meeting notes, agendas, action items |
| `05-Decisions` | ADRs (Architecture/Any Decision Records) |
| `06-People` | Team directory, roles, contact info |
| `07-Research` | Market research, competitor analysis, user insights |
| `08-Templates` | Reusable templates for all of the above |
| `09-Archive` | Completed/deprecated docs moved here |

## Conventions

- File names: `YYYY-MM-DD - Title.md` for dated docs, `Title.md` for evergreen
- Use `[[wiki links]]` to connect related docs
- Tag with `#status/draft`, `#status/review`, `#status/final`
- Every decision gets a record in `05-Decisions`

## Access

All agents and board members have read/write access. Location: `/opt/obsidian-vault/`
